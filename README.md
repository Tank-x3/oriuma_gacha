# オリウマ ガチャシミュレーター (Ori-uma Gacha Simulator)

「オリウマ（オリジナルウマ娘）」のガチャ演出を再現・体験できる、ブラウザベースのWebツールです。
テキストベースのリスト表示形式を採用し、ドキドキする「昇格演出」や「期待度示唆」を楽しめます。

## 🐴 主な機能 (v0.01)

* **単発 / 10連ガチャ:** ボタンひとつでシミュレーションを実行できます。
* **こだわりの演出:**
    * 結果に応じたゲートテキストの変化（「Eclipse first...」など）。
    * クリックで1枚ずつめくる「順次開示」形式。
    * ★1→★2→★3といった「昇格演出」の実装。
* **キャラクター管理:** `data.js` を編集することで、誰でも簡単にキャラクターを追加・削除できます。
* **スマホ対応:** PC・スマートフォンの両方で動作します。

## 🚀 使い方

### 1. ツールを遊ぶ
（GitHub PagesのURLが決まったらここにリンクを貼ってください）にアクセスしてください。
「10連で引く」ボタンを押すと演出が始まります。画面をクリック（タップ）して結果を進めてください。

### 2. 自分の環境で動かす / カスタマイズする
このリポジトリをダウンロード（Clone）し、`index.html` をブラウザで開くだけで動作します。
