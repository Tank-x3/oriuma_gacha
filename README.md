# オリウマ ガチャシミュレーター (Ori-uma Gacha Simulator)

コミュニティ発のオリジナルウマ娘（オリウマ）たちが登場する、Webブラウザ用ガチャシミュレーションツールです。
実際のソーシャルゲームのような演出と確率で、オリウマたちとの出会いを楽しめます。

👉 **[Play Now (ここをクリックして遊ぶ)](https://tank-x3.github.io/oriuma_gacha/)**

## 🐴 主な機能

### 1. 本格的なガチャ演出
* **ゲート演出:** ガチャ開始時のテキスト（「Eclipse first...」など）によって期待度が変化します。
* **昇格演出:** 銀色の星が金色へ、さらには虹色へ…！ 最後まで結果が分からないドキドキ感を再現しています。
* **★3セリフ表示:** ★3キャラクターが登場すると専用セリフが表示されます。2段階の演出でじっくり楽しめます。
* **シークレット枠:** 運が良ければ、通常では出会えない「シークレット（★4）」が登場することも…？

### 2. ピックアップガチャ
* **排出確率アップ:** 特定のキャラクターがピックアップ対象に設定されている場合、そのキャラクターの排出確率がアップします。
* **お知らせで確認:** ピックアップ中のキャラクターは画面上部のお知らせバナーでお伝えしています。

### 3. スキップ機能
* **通常スキップ:** 演出の途中で「SKIP」ボタンを押すと、残りの結果を一括表示できます。★3やシークレットが出た場合はセリフ演出で一度止まります。
* **止まらずスキップ:** SKIPボタンの下にある「止まらずスキップ」チェックボックスをONにすると、★3演出でも止まらず最後まで一気にスキップできます。お急ぎの方に便利です。

### 4. 便利機能
* **リトライ:** 結果画面の「もう一度引く」ボタンで、直前と同じ回数（単発 or 10連）ですぐにもう一度ガチャを引けます。
* **画像保存・SNSシェア:** 結果画面の「画像で保存」ボタンで、ガチャ結果のスクリーンショットを画像として保存できます。SNSでの共有にどうぞ！
* **履歴機能:** 過去100回分の結果を自動で保存。「履歴を見る」ボタンからいつでも振り返ることができます。
* **実装数・確率表示:** 現在登録されているキャラクター数と提供割合をリアルタイムで表示しています。

### 5. 統計・記録
* **トリップメーター:** 好きなタイミングでリセットできるガチャ回数カウンターです。「ここから何回引いたか」を手軽に把握できます。
* **統計モーダル:** 「📊 統計」ボタンから、累計ガチャ回数・レアリティ別の排出実績・ピックアップ排出の詳細・最頻出キャラTOP3などをまとめて確認できます。
* **コンプリート記録:** 全キャラクターを引き当てる「コンプリートチャレンジ」機能です。ノーマルコンプリート（★1〜★3）とフルコンプリート（★4含む）の2段階があり、達成記録が残ります。新キャラが追加されると自動で検出してお知らせします。

### 6. 配信モード
* **配信セッション用の一時統計:** Discord等でのガチャ配信中に、その配信セッション限定の統計を記録・表示できるモードです。
* **使い方:** 画面の「📡 配信モード」エリアで `[開始]` を押すと配信モードがONに。配信中のガチャ結果は通常の統計と並行して一時統計にも記録されます。 `[統計]` で配信中の統計を確認、`[終了]` で配信統計のサマリーを表示して終了できます。
* **ページを閉じても安心:** 配信モード中にページをリロードしたりブラウザを閉じたりしても、次回アクセス時に配信モードが継続されます。

### 7. キーボードショートカット
* **PCでのキーボード操作:** PCではキーボードだけでガチャを操作できます。`1` で単発、`0` で10連、`Space` / `Enter` で次へ進む、`S` でスキップなど。
* **ショートカット一覧:** `H` キーを押すとショートカット一覧が表示されます。初めての方はまず `H` キーを押してみてください。

### 8. スマートフォン対応
* PCはもちろん、スマートフォンの縦画面でも快適に動作します。

## 🚀 遊び方

1.  **ガチャを引く:**
    * 公開URLにアクセスし、「単発で引く」または「10連で引く」ボタンを押してください。
2.  **結果を見る:**
    * 画面をタップ（クリック）すると、1枠ずつ結果が開示されます。
    * お急ぎの場合は「SKIP」ボタンで結果を一括表示できます。
3.  **もう一度引く / 画像で保存:**
    * 結果画面から「もう一度引く」で同じ回数でリトライ、「画像で保存」でスクリーンショットを保存できます。
4.  **統計や履歴を確認する:**
    * 画面下部の「📜 履歴」で過去の獲得履歴、「📊 統計」で累計データやコンプリート進捗を確認できます。
5.  **キーボードで操作する（PC）:**
    * `H` キーでショートカット一覧を表示。キーボードだけでサクサク操作できます。

## ❓ よくある質問

* **Q. 新しいキャラクターはいつ追加されますか？**
    * A. 実装の反映は不定期です。ただし、オリウマレースでの勝利など、記念すべきタイミングが直近にあった場合はHOTなうちに追加されるかもしれません。
* **Q. 提供割合（確率）はどこで確認できますか？**
    * A. 画面上部に実装キャラクター数と提供割合が表示されています。また、「📊 統計」から自分の実績確率も確認できます。
* **Q. 配信モードって何ですか？**
    * A. Discordなどでガチャ配信をする際に、その配信中だけの統計を記録できる機能です。「📡 配信モード」の `[開始]` ボタンから使えます。配信が終わったら `[終了]` で統計のサマリーを確認してクリアできます。
* **Q. 統計データをリセットしたいです。**
    * A. 統計モーダル内の「データ削除」ボタンからリセットできます。トリップメーターはTrip表示横のリセットボタンで個別にリセットできます。
* **Q. 自分のオリウマも追加してほしい！**
    * A. オリウマ娘・二次三次創作喫茶「making duo」で申請してください。現行スレッドは[こちら](https://www.88-ch.com/thread/202602080002)。
